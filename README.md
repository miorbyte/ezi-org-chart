# 📊 Ezi Org Chart Generator

Aplikasi web mesra telefon untuk membina carta organisasi dengan pantas.

## 🚀 Fungsi Utama
- **Gambar Passport**: Boleh muat naik gambar terus ke dalam carta.
- **Offline Mode**: Simpan dan buka data dari fail `.json` di peranti anda.
- **Custom BG**: Tukar warna latar belakang carta.
- **Cetak**: Eksport carta ke format PDF.

## 🛠️ Cara Guna
1. Klik `index.html`.
2. Masukkan maklumat staf.
3. Klik "Simpan" untuk simpan data ke phone/PC.
4. Klik "Cetak" untuk dapatkan fail PDF.
